A propos de moi

Je m'appelle Stéfen j'ai 19ans. J'ai toujours été passionné par l'informatique depuis le collège. J'ai pris la spécialité NSI au Lycée : j'ai pu y apprendre beaucoup de Python, du HTML, du CSS ainsi qu'une base de Javascript et SQL. Je me suis donc orienté vers le developpement web après le lycée ce qui m'a amené dans le centre de formation La Manu où j'ai pu y apprendre de nouvelles choses qui comprends : le HTML et BOOSTRAP, le CSS, le Javascript, le PHP, puis le SQL. Aujourd'hui j'ai donc des connaissances solides dans les languages que j'ai cités précedemment, c'est pour cela que j'aimerais m'orienté, à la suite de cette formation se terminant le 14/04/2023, vers une alternance afin de construire d'avantages de connaissances ainsi que d'être dans de vrais circonstances d'entreprise. Je suis également à la recherche d'un stage d'une semaine pour me faire une première idée de ce que peux ressembler le travail en entreprise.

Merci d'avoir lu, en espérant avoir une réponse !
